## Quick Answers

### Why did my requisition reroute?
See [Requisition Rerouting](reroute-rules.md) for complete details. Short answer: amount exceeded tolerance, approval simulation found different approvers, or a monitored field changed.

### How do I set up SSO with Azure?
See [SSO Azure Setup](sso-azure-setup.md) for step-by-step instructions.

### Can someone approve their own requisition?
Controlled by `USE_APP_AMOUNT_OWN_REQS` setting. See [Approvals FAQ](approvals.md).
