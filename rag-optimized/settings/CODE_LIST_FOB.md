# CODE_LIST_FOB

## Description
code_fldname

This is a pointer to the code_mstr field name (code_fldname) value to be used for the Header FOB selection list and validation. 

You can also use the prefix "LIST:" followed by a comma-separated list of values. This will tell iPurchase that the specified list is to be used. 

Example: LIST:EA,BX,PK   

or 

Separate each entry with a colon to have both a code and name: LIST:EA:EACH,BX:Box,PK:Pack 


## Default Value
`List:ISS:US,Supplier:Supplier`

System Setting
