# AUTO_RECEIVE - iPurchase System Setting

**Category:** Receiving

TRUE | FALSE. If TRUE, automatically creates receipt transactions when purchase order is created. Timing controlled by AUTO_RECEIVE_TIME setting.

### Valid Values

| Value | Behavior |
|-------|----------|
| **TRUE** | Enables this feature |
| **FALSE** | Disables this feature |

### How It Works

See the description above for details on how this setting affects system behavior.

### Setting Details

| Property | Value |
|----------|-------|
| **Setting Name** | AUTO_RECEIVE |
| **Category** | Receiving |
| **Owner** | Purchasing |
| **Default Value** | FALSE |

### How to Query

```sql
SELECT pf_chr1 FROM PUB.pf_mstr
WHERE pf_us_id = 'SYSTEM' AND pf_group = 'DEFAULT' AND pf_attr = 'AUTO_RECEIVE'
```
