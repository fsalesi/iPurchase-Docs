# USE_IPURCHASE_REQ_NBR

## Description
By default, iPurchase use the same mechanism to assign requisition numbers as QAD, namely the work order control file's next work order number. If this setting is true, then the next number will be found using a separate iPurchase sequence.

## Default Value
`TRUE`

System Setting
