# CODE_LIST_H_SHIPVIA_XREF_[VENDOR NUMBER] 

## Description
This setting defines a cross-reference between the selected iPurchase ship via, and the code or description that the vendor needs to see on their electronic purchase order. Only applies to Punchout Purchase Orders.

The [Vendor Number] needs to be the actual vendor number that this cross reference is valid for.

The format should be LIST:UPS:UPS Ground,Fedx,Fedex 3 day

Where UPS is the value for this ship via in iPurchase or QAD, and UPS Ground is the value the vendor requires.



## Default Value
`(not set)`
