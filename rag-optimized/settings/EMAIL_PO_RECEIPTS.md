# EMAIL_PO_RECEIPTS - iPurchase System Setting

**Category:** Email Configuration

TRUE | FALSE. If TRUE, sends email receipts/confirmations when purchase orders are created or modified.

### Valid Values

| Value | Behavior |
|-------|----------|
| **TRUE** | Enables this feature |
| **FALSE** | Disables this feature |

### How It Works

See the description above for details on how this setting affects system behavior.

### Setting Details

| Property | Value |
|----------|-------|
| **Setting Name** | EMAIL_PO_RECEIPTS |
| **Category** | Email Configuration |
| **Owner** | Purchasing |
| **Default Value** | FALSE |

### How to Query

```sql
SELECT pf_chr1 FROM PUB.pf_mstr
WHERE pf_us_id = 'SYSTEM' AND pf_group = 'DEFAULT' AND pf_attr = 'EMAIL_PO_RECEIPTS'
```
