# ALLOW_EMT

## Description
Comma Separated list of User ID's or Group ID's who are allowed to convert sales orders to PO's.  Asterisk indicates everyone, a blank indicates no one. 

This setting is very complicated, please contact ISS Group before making changes to this setting.


## Default Value
`*`

System Setting
