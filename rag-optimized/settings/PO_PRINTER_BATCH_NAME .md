# PO_PRINTER_BATCH_NAME 

## Description
This setting allows the administrator to set the Queue that the report will be processed on.  ex: "POPrint"

QAD EE 2012 and above allows the ability to print fancy purchase orders.

Requirements:
  Names QAD user.
  Settings QAD_AUTHENTICATION_USER and QAD_AUTHENTICATION_PASSWORD must be set.
  The user specified in QAD_AUTHENTICATION_USER must be setup in QAD with a role of "rptAdmin"

## Default Value
`(not set)`

System Setting
